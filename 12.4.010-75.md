# 2. Классификация изделия

| Тип | Описание конструкции                   | Особенности                   |
| --- | -------------------------------------- | ----------------------------- |
| Б   | С втачным напалком                     | Классическая конструкция      |
| В   | Напалок цельнокроенный с нижней частью | Упрощённая конструкция        |
| Г   | Напалок сбоку (универсальный)          | Для правой и левой руки       |
| Д   | Два напалка (большой + указательный)   | Повышенная функциональность   |
| Е   | Удлинённые с крагами                   | Стягивание лентой, наладонник |

Дополнительно:

| Параметр | Значение                                       |
| -------- | ---------------------------------------------- |
| Усиления | С накладками или без                           |
| Размеры  | 0, 1, 2, 3                                     |
| Накладки | Форма и расположение определяются потребителем |
